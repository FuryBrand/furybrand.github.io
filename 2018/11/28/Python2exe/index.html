Python-将python文件打包成exe文件

Posted by Steve on November 28, 2018
  • 1. 安装PyInstaller
  • 2. 打包成exe
  • 3. 出现过的问题

今天写了一个小工具，但是想要让大家用的话，搞环境啥的太烦了，本来就是个小工具嘛。所以搜了下如何将python打包发布成exe应用。

1. 安装PyInstaller

用pip就可以了。



1



pip install PyInstaller

2. 打包成exe

使用这个命令就可以打包exe了，pyinstaller -F -c 小工具.py。

一般常用的参数是：

  • -w 或 --windowed：这个选项用于告诉 PyInstaller 打包的应用程序不应该带有控制台窗口。这对于图形用户界面（GUI）应用程序很有用。如果程序是命令行工具，那么不应该包含这个选项。
  • -F 或 --onefile：这个选项告诉 PyInstaller 将所有的东西打包成一个单独的可执行文件。默认情况下，PyInstaller 会创建一个包含可执行文件和一个文件夹的分布，文件夹中包含程序运行所需的所有依赖项。使用 -F 选项后，所有的内容都会被打包进一个文件，这使得分发程序更加简单，但启动时间可能会稍长一些，因为在运行之前需要先解压。
  • -c 或 --console，--nowindowed：这个选项是默认设置，它告诉 PyInstaller 打包的应用程序应该带有控制台窗口。通常不会在同一个命令中同时使用 -w 和 -c，因为它们是相互矛盾的。如果同时使用 -w 和 -c，-w 会覆盖 -c 的效果。

3. 出现过的问题

打包过程中出现了UnicodeDecodeError: 'gbk' codec can't decode byte 0xab in position 160这样的错误信息。

原因时打包的python文件含有中文字符。所以两个解决方案，要么改成英文名；要么修改PyInstaller的源码，报错部分是winmanifest.py文件的1075行，修改为with open(filename, 'rb') as f:就可以了。
